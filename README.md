# `morrsse`

BSD's `morse` but with more `rs`!

## Install

```bash
cargo install morrsse
```

## Usage

```bash
# As arguments
morrsse "one phrase" "another phrase" "SMS" "galaga"
# As STDIN
echo "one phrase" "another phrase" "SMS" "galaga"  | morrsse
# See help for more options
morrsse --help
```
